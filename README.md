## CSGO 饰品自由议价

> 得润(need.run)官方脚本，用于 buff、悠悠有品、igxe、c5 饰品直达自由交流区，无惧敏感词

### 使用教程

1. 请确保浏览器已经安装 tampermonkey，这一步不会请百度
2. 安装本脚本
3. 访问平台，可以在在售列表的每一件饰品中，看到**联系**按钮
4. 点击**联系**即可跳转到 needrun 进行议价交流

### 支持平台

-   buff
-   悠悠有品
-   igxe
-   c5
